# Convert becauseR equations to ggm DAG adjacency matrix

Convert becauseR equations to ggm DAG adjacency matrix

## Usage

``` r
equations_to_dag(equations, exclude_vars = NULL)
```

## Arguments

- equations:

  List of formulas

- exclude_vars:

  Character vector of variable names to exclude (e.g., grouping
  variables)

## Value

Named adjacency matrix in ggm format
