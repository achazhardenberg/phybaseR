# Transform Graph for D-Separation

Modules implement this to rename variables in the causal graph to match
their latent structure.

## Usage

``` r
transform_graph_for_dsep(family, equations, ...)
```

## Arguments

- family:

  The S3 family object.

- equations:

  The list of structural formulas.

- ...:

  Additional arguments.

## Value

The transformed list of formulas.
