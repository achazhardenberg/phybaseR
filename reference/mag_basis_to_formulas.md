# Convert MAG basis set to becauseR formula format

Convert MAG basis set to becauseR formula format

## Usage

``` r
mag_basis_to_formulas(basis_set, latent_children = NULL)
```

## Arguments

- basis_set:

  Basis set from basiSet.mag()

- latent_children:

  Optional character vector of variables that are direct children of
  latents

## Value

List of formulas with test_var attribute
